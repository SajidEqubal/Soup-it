# Soup-it

An android app for  mantel health's


App Link - https://play.google.com/store/apps/details?id=com.shadspace.soup_it 


### Features : 
* Good News
* Play games
* Bolgs
* Short Story
* Write & Earn
* Daily Quotes


### Contains : 
* Login/SignUp
* Animaion 
* About & Contacts 
* Events (ongoing/past/upcoming)
* View Submission (Short Story) 
* Dark Mode
